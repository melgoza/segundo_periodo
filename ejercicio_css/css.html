CSS

¿Qué es css?

Css es un lenguaje de programación que tiene como función darle el diseño y todo lo que tiene que ver con lo visual al programar. Para vincular un archivo css con uno html, se debe usar una etiqueta "link".

¿Qué es una referencia por palabra clave?

Una referencia por palabra clave es cuando quieres que mediante css, todas las etiquetas en tu archivo de html tengan alguna propiadad, como por ejemplo que todos los "articles" tengan un fondo azul.

¿Qué es una referencia por clase?

Esto se usa cuando quieres que ciertas etiquetas de un grupo de etiquetas tengan alguna característica. Por ejemplo, que todas las etiqueta "article" tengan el fondo azul, pero de todas esas, quieres que solo algunas tengan el color de letra negro. Las clases son más espeíficas que una referencia por palabra clave

¿Qué es una referencia por id?

Este tipo de referencia se usa una sola vez por id, es decir que solo puede haber una etiqueta con cierto id. Su función es la misma que la referencia por clase, pero ésta es más específica.

Ejemplo: todas las palabras "computadora" están escritas en etiquetas "p". En un archivo css usé una referenciap or palabra clave a la etiqueta "p", donde especifiqué que todas las etiquetas de este tipo tuvieran un fondo azul, usando una referencia por clase, solamente 3 de ellas tienen las letras más grandes, y usando una referencia por id, especifiqué que solo una estuviera alineada al centro.
